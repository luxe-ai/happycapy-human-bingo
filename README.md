# HappyCapy Human Bingo 真人賓果遊戲

🎉 一個有趣的社交破冰遊戲，幫助你認識新朋友！

## 🎮 遊戲說明

這是一個互動式的真人賓果遊戲，參與者需要找到符合每個格子條件的朋友，並讓他們分享真實的故事。完成一條橫線、直線或對角線即可達成Bingo！

### 📱 特色功能

- ✅ **雙語支援**：粵語 / English 隨時切換
- ✅ **移動端友好**：完美適配手機、平板和桌面設備
- ✅ **翻卡互動**：點擊卡片輸入朋友名字
- ✅ **防重複**：同一局遊戲中每個名字只能使用一次
- ✅ **洗牌功能**：可以重新洗牌開始新一局
- ✅ **實時檢查**：即時檢查是否達成Bingo

### 💰 獎勵機制

- 🎁 **第一名獎勵**：第一個達成Bingo並截圖的隊伍（限時5分鐘），每人獲得額外 **500 credit**
- 🏆 **最多Bingo獎勵**：達成Bingo數量最多的隊伍，每人獲得額外 **200 credit**
- ✨ **參與獎勵**：其他達成Bingo的隊伍，每人都可獲得 **200 credit**

**重要提醒**：每個格子中的名字需要本人介紹真實故事，不可以瞎編！

## 🚀 如何使用

### 線上遊玩

直接訪問 GitHub Pages 部署的遊戲：
```
https://[你的用戶名].github.io/[倉庫名]/
```

### 本地運行

1. 克隆此倉庫
```bash
git clone https://github.com/[你的用戶名]/[倉庫名].git
```

2. 直接在瀏覽器中打開 `index.html` 文件即可遊玩

無需安裝任何依賴或運行服務器！

## 📋 遊戲規則

1. 點擊卡片翻轉，輸入符合條件的朋友名字
2. 已標記的卡片再點一下可以取消標記
3. 完成橫線、直線或對角線即可Bingo
4. 中間格子是免費格，自動標記
5. 每個格子中的名字需要介紹真實故事，不可以亂編

## 🛠️ 技術特點

- 純HTML + CSS + JavaScript 實現
- 無需任何框架或依賴
- 響應式設計，支持所有設備尺寸
- 使用現代CSS動畫效果
- 本地存儲遊戲狀態

## 📦 部署到 GitHub Pages

1. 創建新的 GitHub 倉庫
2. 推送代碼到倉庫
3. 在倉庫設置中啟用 GitHub Pages
4. 選擇 `main` 分支作為源
5. 訪問你的遊戲鏈接！

## 🎨 定制

你可以輕鬆修改 `index.html` 中的內容來定制遊戲：

- **修改卡片內容**：編輯 `prompts` 數組
- **更改語言文本**：編輯 `i18n` 對象
- **調整樣式**：修改 `<style>` 標籤中的CSS
- **修改配色**：更改 `:root` 中的CSS變量
